# Ministerio Azure

Esta carpeta representa la infraestructura del Ministerio desplegada como si estuviera en Azure AKS.

Componentes:

- `api-ministerio` con 2 replicas.
- `postgres-ministerio` con PVC.
- Cluster local `kind-cloud-ministerio-azure`.

La red blockchain con Geth se agrega en Semana 7.
