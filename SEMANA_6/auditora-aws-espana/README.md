# Auditora AWS Espana

Esta carpeta representa la infraestructura de la universidad auditora desplegada como si estuviera en AWS EKS en Espana.

Componentes:

- `api-auditora` con 2 replicas.
- `postgres-auditora` con PVC.
- Cluster local `kind-cloud-auditora-aws`.

La red blockchain con Geth se agrega en Semana 7.
