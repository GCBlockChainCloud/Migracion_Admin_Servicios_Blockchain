# UTPL GCP

Esta carpeta representa la infraestructura de la universidad emisora desplegada como si estuviera en Google Cloud GKE.

Componentes:

- `api-universidad` con 2 replicas.
- `postgres-universidad` con PVC.
- Cluster local `kind-cloud-utpl-gcp`.

La red blockchain con Geth se agrega en Semana 7.
