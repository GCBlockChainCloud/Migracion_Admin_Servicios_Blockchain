# Ganache vs Geth

Ganache fue usado en Semanas 3, 4 y 5 para aprender contratos inteligentes, transacciones y conexion desde APIs.

Geth se usa en Semana 6 porque el objetivo cambia: ahora se quiere desplegar infraestructura blockchain distribuida.

| Tema | Ganache | Geth |
| --- | --- | --- |
| Uso principal | Desarrollo local rapido | Nodo Ethereum real |
| Multi-nodo | No es su foco | Soportado |
| Peering P2P | Limitado | Nativo |
| Persistencia | Simple | Realista |
| Kubernetes | Util para pruebas | Mejor para StatefulSet y PVC |
| Multi-cloud | Poco representativo | Adecuado |

Mensaje para clase:

```text
Ganache sirvio para aprender. Geth sirve para simular una red blockchain privada real.
```
